# Tabelog Google Map Extension

食べログのページにGoogle Mapの評価・レビュー情報を表示するChrome拡張機能

## 背景・課題

飲食店を選ぶ際、多くの人が食べログとGoogle Mapの両方を確認している。
しかし、2つのサービスを行き来するのは手間がかかる。

## 解決策

食べログのページを閲覧中に、同じ店舗のGoogle Map評価を自動で取得・表示する
Chrome拡張機能を開発する。

## 主な機能

- [ ] 食べログの店舗ページで店名・住所を自動取得
- [ ] Google Places APIで該当店舗を検索
- [ ] Google Mapの評価（星・レビュー数）をページ内に表示
